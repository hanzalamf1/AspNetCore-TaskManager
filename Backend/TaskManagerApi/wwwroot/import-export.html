İçe Aktarma

Harici kaynaklardan görev verilerinizi içe aktarın. Desteklenen formatlar: CSV, JSON, Excel.

Dosya Seçin veya Sürükleyin
CSV, JSON veya Excel dosyaları desteklenir
İçe Aktarma Kuralları
  • CSV dosyaları için: Başlık satırı gerekli (title, description, dueDate, priority)
  • JSON dosyaları için: Görev dizisi formatında olmalı
  • Excel dosyaları için: İlk satır başlık olarak kullanılır
  • Maksimum dosya boyutu: 10MB
